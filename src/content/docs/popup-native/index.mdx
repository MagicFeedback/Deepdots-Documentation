# Deepdots Popup SDK

Deepdots Popup SDK is a Kotlin Multiplatform SDK that lets teams display in-app popups and launch surveys across Android and iOS from one shared implementation.

## Business value

- Reduce time to launch in-app feedback and engagement flows
- Validate mobile experiences with included Android and iOS demo apps
- Manage popup campaigns centrally in Deepdots, delivered to the app at runtime

## What problem it solves

- popups delivered from Deepdots at runtime
- trigger-based display logic
- language and path segmentation
- embedded survey rendering through MagicFeedback
- popup lifecycle events for analytics and orchestration

## Correct naming

- Commercial name: **Deepdots Popup SDK**
- Repository name: `deepdots-popup-mobile-native-sdk`
- Android package: `com.deepdots.sdk:shared-android`
- Swift Package manifest: `DeepdotsSDK`
- Swift product name: `DeepdotsSDK`
