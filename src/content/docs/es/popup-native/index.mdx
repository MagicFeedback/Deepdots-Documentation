# Deepdots Popup SDK

Deepdots Popup SDK permite mostrar popups y lanzar encuestas dentro de apps Android e iOS desde una base compartida de Kotlin Multiplatform. El repositorio incluye demos nativas, distribución para Android e iOS y soporte para segmentación, triggers y eventos de ciclo de vida.

## Valor para negocio

- Reduce el tiempo para activar campañas o recoger feedback in-app
- Facilita validaciones internas con demos ya incluidas en el repo
- Permite separar el control remoto de campañas con **Server Mode** del control embebido con **Client Mode**
