# Client Mode

`Mode.Client` usa popups definidos localmente en `PopupOptions.popups`.

## Cuando encaja mejor

- validacion local
- catalogo pequeno y estable
- casos donde no se quiere depender del fetch remoto de definiciones

## Diferencia principal

En este modo la definicion del popup viaja con la app; en `Server Mode` llega desde backend.
