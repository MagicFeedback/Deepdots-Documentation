# Quickstart

Popup-definitioner kommer altid fra backend. Demoapps initialiserer blot SDK'et med en gyldig `publicKey` og lader det hente popups i runtime.

## Grundtrin

1. Initialiser `DeepdotsPopups`
2. Angiv `publicKey`
3. Sæt `provideLang`
4. Aktiver `autoLaunch`
5. Kald `setPath(...)`
6. Tilfoej platformskontekst via `attachContext(...)`

## Backend-miljø

`environment` styrer hvilken backend SDK'et taler med (`Environment.Production`
eller `Environment.Development`). Standard er `Production`. Efter initialisering
kan værtsappen læse den read-only `environment`-property på `DeepdotsPopups` for
at logge eller tjekke det aktive miljø.

## Lokaliserede knapper

`provideLang` bruges også til at lokalisere SDK'ets standard-knaptekster
(Send / Cancel / Start survey / Complete survey / Back). Understøttede sprog:
**en, es, da, no, sv, fi, zh-CN**. Regionsvarianter (`da-DK`, `nb-NO`, `zh-Hans`)
accepteres. Ukendte sprog falder tilbage til engelsk. Etiketter sat på
`PopupDefinition.actions` fra backend har altid forrang.
