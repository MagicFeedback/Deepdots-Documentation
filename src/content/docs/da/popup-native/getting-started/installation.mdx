# Installation

Offentlige integrationsspor i repoet:

- Android: `com.deepdots.sdk:shared-android`
- iOS: `DeepdotsSDK` med Swift-produktet `DeepdotsSDK`

## Android

```kotlin
dependencies {
    implementation("com.deepdots.sdk:shared-android:0.3.0")
}
```

## iOS

- SPM package-navn: `DeepdotsSDK`
- Swift-produkt: `DeepdotsSDK`
- Minimum iOS i repoet: `13`
