# Troubleshooting

- Kontroller `publicKey`, `attachContext`, `setPath` og `autoLaunch`, hvis popups ikke vises.
- Kontroller at host-appen sender scroll og custom events videre til SDK'et.
- Kontroller bundling af `magicfeedback-sdk.browser.js`, hvis survey-rendering fejler.
