# Client Mode

`Mode.Client` bruger lokale `PopupDefinition`-objekter i stedet for remote fetch af definitioner.

Det passer bedst til:

- lokal validering
- en lille og stabil popup-katalog
- flows hvor backend-styring ikke er noedvendig endnu
