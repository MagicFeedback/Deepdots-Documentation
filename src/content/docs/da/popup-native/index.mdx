# Deepdots Popup SDK

Deepdots Popup SDK er et Kotlin Multiplatform SDK til at vise popups og starte surveys i Android- og iOS-apps. Repositoriet indeholder native demoapps, distributionsflow for Android og iOS samt understottelse af triggers, segmentering og lifecycle-events.

## Forretningsvaerdi

- Giver teams en hurtigere vej til in-app feedback og aktiveringsflows
- Understotter intern validering via de medfoelgende demoapps
- Giver et valg mellem backend-styret **Server Mode** og lokalt defineret **Client Mode**

- Synligt produktnavn: **Deepdots Popup SDK**
- Android package: `com.deepdots.sdk:shared-android`
- SPM package-navn: `DeepdotsSDK`
- Swift-produkt: `DeepdotsSDK`
